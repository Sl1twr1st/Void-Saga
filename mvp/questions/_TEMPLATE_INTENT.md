
## Maturity Pipeline

```
Candidate Intent   ← baru diusulkan, < 3 sessions
        │
        ▼
Supporting Evidence ← 3–10 sessions, pola mulai terlihat
        │
        ▼
Working Intent     ← 10+ sessions, continue rate stabil, digunakan di interview
        │
        ▼
Adopted            ← digunakan di luar Bab 00, bertahan di konteks berbeda
```

Tidak ada "Validated." Sama seperti invariant — jangan bilang confirmed.
"Working" dan "Adopted" jujur tentang ketidakpastian.
