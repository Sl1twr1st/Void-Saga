# Void Saga

Most fictional universes are documented.  
Void Saga is executable.

Void Saga is not a novel.  
It is not a lore wiki.  
It is not a character database.  
**It is a Narrative Operating System.**

Instead of documenting fictional rules, Void Saga compiles them into executable runtime constraints.

Every character is a runtime.  
Every relationship is a contract.  
Every narrative is compiled before it is allowed to exist.

---

## The Problem

Large language models are excellent storytellers.  
They are also excellent at breaking stories.

A single prompt can accidentally erase years of worldbuilding:

- characters forget their core wounds
- relationships collapse
- canon drifts
- lore contradicts itself
- emotional logic disappears

Traditional fictional universes solve this with documentation.  
Documentation depends on humans remembering it.  
LLMs don't.

---

## The Idea

Treat a fictional universe like software.

Instead of writing lore that authors are expected to remember, compile the universe into executable structures:

- Runtime
- Constraint
- Contract
- Invariant
- Forbidden Behavior

Then execute those rules before any story is generated.

## Live Demo

Generate a canon-compliant scene:

```bash
python3 VOID_SAGA_UNIVERSE/apps/compiler/compiler.py \
  VOID_SAGA_UNIVERSE/apps/engine/scenarios_v2/test_niuniu_sevraya_orbit.json \
  --live
```

Output — a 3420-character narrative generated by Claude, post-validated through the constraint engine:

```
─── GENERATED NARRATIVE ───────────────────────────────
The room holds its breath. NiuNiu watches from the corner—not hiding, just 
positioned. The distance between them is exactly what it needs to be.

Sevraya's lighter clicks. Once. Twice. The flame catches. She draws, and the 
smoke rises in patterns that almost look like writing before they dissolve.

'Aku yang tinggal,' she says. I stay.

The words land flat. Administrative. Not a plea, not a promise. A statement of 
fact, the way someone might say the Grid still holds or the Void still erases. 
Just something true happening in the room.
─── END ───────────────────────────────────────────────
```

Post-gen validation: **PASS** (canon score 1.0).

Attempt to violate canon:

```bash
python3 VOID_SAGA_UNIVERSE/apps/compiler/compiler.py \
  VOID_SAGA_UNIVERSE/apps/engine/scenarios_v2/test_violation_zero_emotional.json
```

Expected result:

```
🛑 GENERATION BLOCKED
   Reason: Hard block: 1 critical violation(s) detected.
   Hard blocks: 1
     🚫 [forbidden_behavior] Forbidden behavior violated:
        Making Zero emotional, warm, or personally attached to anyone.
   Canon score: 0.97
```

Canon score 0.97 — nearly perfect. Still blocked.  
**Claude API was never called.** No tokens spent. No unsafe output generated.
