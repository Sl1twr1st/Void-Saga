## Kenapa Ini Bukan Sekadar Polish

1. **Reader-kernel lahir dari fork tool.** UI yang sama berfungsi sebagai pembaca novel DAN mesin fork. Tidak ada mode switch. Transisi natural: baca → acknowledge → pilih fork atau lanjut.

2. **Setiap elemen UI punya fungsi ganda.** Scroll tracking = system observing progress + consent checkpoint. Segment anchor = daftar isi + canon node. Context bar = informasi sesi + constitutional surface. Gallery = arsip bacaan + residue archive.

3. **Engine jadi terlihat.** "The Archivist translated your fork" bukan gimmick — itu summary dari data yang benar-benar diproses: segment, fork point, laws touched, canon gravity, introduced entities, verdict.

4. **Boot sequence menanam consent.** [Y/N] di awal mencerminkan Opening Void Saga — consent ditawarkan, tapi N pun tetap lanjut. Ini bukan dekorasi. Ini DNA cerita.
