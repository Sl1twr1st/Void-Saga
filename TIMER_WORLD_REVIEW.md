# TIMER_WORLD_REVIEW.md

Working table untuk review temporal semua Timer. Ini **belum** mengubah manifest; isinya adalah pembacaan kerja awal berbasis file Timer.

Catatan metodologis:
- `composed_during` diisi **unknown** kalau tidak ada bukti eksplisit di file Timer/Bab/metadata.
- Resonance pair Bab↔Timer **tidak otomatis** dipakai sebagai bukti bahwa Timer itu ditulis saat Bab pair-nya.
- File ini adalah **review observasional**, bukan keputusan canon. Keputusan canon harus lahir di tahap editorial terpisah sebelum apa pun masuk manifest.

Vocabulary final yang dipakai di file ini:
- `mode`: `present_event` | `archival_record` | `reflective` | `mixed`
- `temporal_status`: `sequential` | `retroactive` | `recursive` | `atemporal` | `uncertain`
- `certainty`: `exact` | `approximate` | `relative` | `unknown`
- `confidence`: `high` | `medium` | `low`
